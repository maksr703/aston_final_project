# 🚀 Learn Aston Step 1 — Final Team Project

---

## 📌 Описание проекта

Проект представляет собой Java-приложение для работы с пользователями и их данными.

Основной функционал включает:
- ручной ввод пользователей
- генерацию пользователей
- обработку и чтение данных из файлов
- сортировку пользователей
- сравнение email-адресов
- параллельный подсчет данных
- использование кастомных коллекций и структур данных
- юнит-тестирование

Проект написан на Java с использованием Gradle и покрыт unit-тестами (JUnit 4 и JUnit 5).

---

## 🧱 Основной функционал

### 👤 Работа с пользователями
- создание пользователей через Builder
- валидация email, имени и пароля
- генерация случайных пользователей

### 📂 Работа с файлами
- чтение данных из файла
- фильтрация пустых и некорректных строк
- преобразование строк в объекты `User`
- запись отсортированного списка в файл (режим добавления данных)

### 🔀 Сортировка и сравнение

- алгоритм сортировки: **QuickSort**
- сортировка пользователей по кастомным правилам

#### 📧 EmailComparator
- сравнение по домену
- при равенстве домена — сравнение по local-part

#### 🔐 UserPasswordSorter
- чётные пароли сортируются по возрастанию
- нечётные пароли остаются на своих позициях

---

### ⚡ Параллельная обработка
- подсчет пользователей по email-домену

---

### 📦 Кастомные структуры данных
- `CustomCollection`
- `CustomArrayList`

---

## 🧪 Тестирование

- JUnit 4 (основная часть тестов)
- JUnit 5 (Jupiter) — используется частично (1 тест: RandomUserGeneratorTest)
